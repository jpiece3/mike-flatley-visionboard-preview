Behind-the-Scenes: Profile Matching Algorithm

Technical

Matching Logic (tag-based overlap):

  1. User completes 6 steps ‚Üí generates combined tag array
  2. System compares user tags to each profile's trigger_tags
  3. Calculates overlap ratio: matching_tags / total_profile_trigger_tags
  4. Profile with highest overlap ratio wins (threshold: 0.5)
  5. If no profile meets threshold, fallback to "The Nester" (largest Chester County segment)

Example Match:

User selects: Modern Luxury + Quiet Suburbs + 4 Bedrooms + Smart Home + Natural Light + $750K-$1M + 3-6 Months

User tags: modern, luxury, clean-lines, contemporary, suburban, quiet, large, family, smart-home, natural-light, budget-premium, 6-months, planning

The Minimalist trigger tags: modern, clean-lines, urban, loft, smart-home, natural-light

Overlap: 4 of 6 match (modern, clean-lines, smart-home, natural-light) = 67% ‚Üí Match!

Qualification Signal (Backend Only)

Not Shown to User

Used for email sequence routing:

Formula:

urgency_score * 0.6 + budget_score * 0.4

Urgency Scoring

  • Hot (1.0) immediate, urgent, active-search
  • Warm (0.6) 3-months, 6-months, planning
  • Cool (0.3) browsing, exploring, next-year

Budget Scoring

  • Hot (1.0) budget-premium, budget-luxury
  • Warm (0.6) budget-mid, budget-upper
  • Cool (0.3) budget-starter

Email Sequence Routing:
