Key Animations

  • Fade In: Opacity 0 → 1, translateY(8px → 0)
  • Scale In: Scale 0.95 → 1, opacity 0 → 1
  • Slide In: translateX(-20px → 0), opacity 0 → 1

Implementation Summary

CSS Variables Setup

All design tokens are defined as CSS custom properties in :root for easy theming and maintenance:

:root {
  /* Colors */
  --color-primary-navy: #1a3a52;
  --color-accent-gold: #d4a574;

  /* Typography */
  --font-heading: 'Playfair Display', serif;
  --font-body: 'Inter', sans-serif;

  /* Spacing */
  --space-4: 1rem;
  --space-6: 1.5rem;

  /* Shadows */
  --shadow-md: 0 2px 4px rgba(26, 58, 82, 0.04);

  /* Transitions */
  --transition-base: 250ms ease-in-out;
}

Component Structure

All components follow a consistent pattern:

  1. Base styles with CSS variables
  2. Modifier classes for variants (e.g., .button--primary, .button--secondary)
  3. Responsive breakpoints using media queries
  4. Accessibility attributes (ARIA, focus states)
  5. Smooth transitions for interactive states

Performance Considerations

  • Font Loading: Use font-display: swap to prevent FOIT
  • Image Optimization: WebP format with fallbacks, lazy loading
  • CSS: Critical CSS inlined, rest loaded async
  • Animations: Use transform and opacity for GPU acceleration
