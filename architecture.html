Match Algorithm

For each profile:
  user_tags = all tags from user's selections across all 6 dimensions
  overlap = count(user_tags INTERSECT profile.trigger_tags)
  ratio = overlap / count(profile.trigger_tags)
  if ratio >= profile.match_threshold (0.5): candidate

Winner = candidate with highest ratio
Ties broken by profile order (first in array wins)
If no candidate exceeds threshold: fallback to The Nester

Limited Chester County inventory. Longer nurture.

Composite Formula

composite = urgency * 0.6 + budget_fit * 0.4

Signal values:
  hot = 1.0
  warm = 0.5
  cool = 0.0

Composite thresholds:
  >= 0.7 = Hot lead (immediate outreach, schedule showing)
  0.3 - 0.69 = Warm lead (nurture sequence, market updates)
  < 0.3 = Cool lead (long-term drip, inspiration content)

Weight rationale: Urgency weighted higher (0.6) because real estate is timing-driven. The Gary Mercer Team's volume model (401 sales/year) means they can serve all budget levels -- urgency is the stronger signal.
