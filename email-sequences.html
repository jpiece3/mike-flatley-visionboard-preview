  • Warm/Cool leads: 7 emails (Welcome + Warm Nurture + Re-Engagement)

Email Design Notes

Voice: Conversational, transparent, no-BS. Every email gives value before asking for anything. CTAs: Primary CTA in every email. Secondary CTA where appropriate. Never more than 2 CTAs per email. Personalization: Profile name and description used throughout. Dynamic content based on selections (neighborhood, must-haves, budget). Unsubscribe Strategy: Explicitly mentioned in re-engagement emails to reduce list fatigue. "No hard feelings" language reduces guilt. Mobile Optimization: Short paragraphs, bullet points, clear CTAs. Designed to be read on phones. Deliverability: Plain text with light HTML formatting. No heavy images. Sender: "Sharra Mercer | The Gary Mercer Team" for recognition.
